# Samra Amir — Personal Portfolio (React + Vite + JavaScript)

This is the JavaScript (no TypeScript) version of the portfolio.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Works out of the box on Vercel, Netlify, and Cloudflare Pages — just point them at this repo. Build command: `npm run build`. Output directory: `dist`.
